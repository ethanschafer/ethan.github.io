Name Assignment

Description: In my first Scratch project, the Name Assignment, I tested out animations with my name. I made my name move on different axis and change size. I also made a reset button and a clear button. The reset button starts the animation over and the clear sets my name to normal.

Concepts Learned: Using event blocks- I used event blocks to start and end the animation Using conditional statements- I used “if statements” to reset the animation Using loops and forever statements- I used loops to make the letters of my name move for an amount of time. Sprites- created sprites using sprite editor Movement- used x and y values to move sprites

Computer Buying Project

Description: In the Computer Buying Project our group would interview someone looking to buy a computer. Next, we would use their questions to find them the ideal computer. Finally, we put it into Scratch and made it a project.

Concepts Learned: Using variables to make more smooth movement Hiding and Showing Sprites- used to go through the animation Changing backgrounds smoothly- used to get to the next area of the animation Speech Bubbles used when a player is near

Final Project

Description: In the final project, I coded a platform game. Everything in it is made independently. I created 20 interactive levels and 4 worlds. Next I coded a character who is affected by gravity. I created a death counter, level counter, and a timer.

Concepts Learned: Using variables to create a gravity effect. Using sensors to detect when a player fails. Using sensors to create a start menu. Using broadcasts to detect when the game starts Automatically changing costumes when a player moves
