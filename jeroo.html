Jeroo Content

Fenced-In Lab

The Fenced-In Lab is the first lab we did. This lab is designed to teach us how Jeroo works by using objects and the environment.

Nets Lab

The Nets Lab is also made to teach us how we can use objects in Jeroo to manipulate the environment.

Diamond Lab

The purpose of this lab is to show us that repetition takes a while. It also teaches us how to manipulate the Jeroo as it spawns in.

The Box Lab

The Box Lab is designed to show us how to make basic method statements to make the code more clean.

The Water Lab

This lab is created with the purpose of teaching us how to use methods to avoid things in the Jeroo's environment, such as water or nets.

The LED Lab

The LED Lab goes more into depth on how we can use methods to clean up code. For this lab, we had to use different methods to create different shapes

Flower Lab

The Flower Lab is designed to make us more comfortable with methods while using manipulated preconditions.

The Escape Lab

The Escape Lab is a challenging lab because we have 4 Jeroos to control, but they are controlled with only one method. It teaches us problem-solving skills

The Pinball Lab

The Pinball Lab is designed to teach us how to use Jeroo methods to avoid water but also loop things.

Flower Lab 2

The second Flower Lab is similar to the first Flower Lab, but this time it has water around the island so we need to use methods to avoid conditions.

Spiral Lab

The Spiral Lab teaches us how to use a single loop to perform tasks in different directions.

The Water Lab

This lab was designed with the intent of combining loops, methods, and if statements to solve a problem

Lockup Lab

This lab shows us how to breakup larger tasks into smaller ones in order to solve a complicated problem
