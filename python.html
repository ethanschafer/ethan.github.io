Python Content

Unit 1

In Unit 1, we learned how to use variables, classes, and methods. In the Variables Lab, we learned to use variables by doing simple calculations. In the Math Tricks lab, we used methods to do more complicated calculations. In the Moving Lab, we created a character using methods and classes.

Unit 2

In Unit 2, we learned how to use if statements, if-else statements, and nested if statements. In the Don't Walk Off Lab, we used if-statements to prevent the person from walking off of the screen. In the Collide Lab, we used if-else statements to prevent the person from going off the screen or touching a wall on the screen. In the Running Man Lab, we used nested if statements to allow a person to jump to a certain height before coming to the ground.

Unit 3

In Unit 3, we learned how to use loops, strings, and nested loops. In the Timer Lab, we used loops to repeat code that counts down. In the Caesar's Cipher Lab, we used loops, strings, and methods to encode a secret language. In the Squares and Circles Lab, we used loops and variables to draw multiple squares and circles. In the Tiles Lab, we created nested loops to create a surface with grass and rocks.

Unit 4

In Unit 4, we learned how to use lists. In the Costume Change Lab, we used lists to make multiple costumes for a sprite, that changes costumes when the sprite moves. In the Changing Ground Lab, we used loops and ifs with lists to create a ground-like background generated with tiles. In the dodgingEnemies lab, we used lists to create a game. In the game, there were enemy sprites moving across the screen, and the goal is to get to the other side.
